# haskell-basics


## Installation

### Windows

1. Scoop

```bash
scoop install main/haskell
```


### Linux

```bash
sudo apt-get install haskell-platform
```

## Mac

```bash
brew install ghc cabal-install
```

## Interactive mode

```bash
ghci
```


## Running Lessons and Examples

```bash
ghc -o <output> <input>
```


Happy Hacking!
